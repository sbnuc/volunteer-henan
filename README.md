# 河南高考志愿填报系统

结合性格测试结果与高考分数，智能推荐适合的专业与院校。数据来源为 EOL 阳光高考 API（2023-2025年真实录取数据）。

## 功能特性

- **5种性格测试**：MBTI、霍兰德职业兴趣、大五人格、职业锚、DISC 行为风格，支持选择至少2个进行综合测评
- **智能专业推荐**：基于性格画像匹配专业方向，输出 Top 推荐专业列表及匹配度
- **精准院校推荐**：结合分数、位次、选科组合，从真实录取数据中筛选可冲刺/稳妥/保底院校
- **保存结果**：一键导出推荐结果为文本，方便后续参考


## 使用流程

1. **选择测试** — 从 5 种性格测试中选择至少 2 个
2. **完成测试** — 依次作答，系统自动分析性格画像
3. **输入分数** — 填写高考分数、省排名位次、首选科目组（物理/历史）及再选科目
4. **查看推荐** — 查看专业推荐和院校推荐结果，可保存为文本

```

├── styles.css                  # 样式表
├── app.js                      # 主逻辑（流程控制、事件绑定）

│   ├── run.py                  # 爬虫入口
│   ├── schools.py              # 院校数据爬取
│   ├── scores.py               # 录取分数爬取

```

## 技术栈

- **前端**：原生 HTML / CSS / JavaScript，无任何框架依赖
- **数据采集**：Python 爬虫（EOL 阳光高考 API）
- **数据规模**：覆盖 1000+ 院校、数千条真实录取分数线记录

## 数据来源

院校及录取数据来自 [EOL 阳光高考](https://gaokao.chsi.com.cn/) 公开 API，包含 2023-2025 年河南省真实录取数据。
